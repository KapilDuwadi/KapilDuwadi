## What I Work On

Power grids are getting complicated fast — solar panels, batteries, EVs, and microgrids are all landing on infrastructure that was designed decades ago for one-way power flow. The software to model, simulate, and manage that complexity is mostly ancient or nonexistent.

I build the tools that help engineers, utilities, and regulators understand what's actually happening on the grid — and what happens next.

Right now that means:
- **Data modeling** — clean, typed, interoperable schemas for grid assets
- **Impact analysis** — what happens to the grid when you add 500 solar installations?
- **AI integration** — using LLMs to make grid data accessible without needing a PhD to query it
- **Full-stack apps** — tools that researchers and operators can actually use
