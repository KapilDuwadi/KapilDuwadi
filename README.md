## ⚡ What I Work On

Power grids are getting complicated fast — solar panels, batteries, EVs, and microgrids are landing on infrastructure designed decades ago for one-way power flow. The software to model, simulate, and manage that complexity is mostly ancient or nonexistent.

I build the tools that help engineers, utilities, and regulators understand what's actually happening on the grid — and what happens next.

Right now that means:
- **🗄️ Data pipelines** — TB-scale time series simulation data, streamed and structured
- **🗺️ GIS conversion** — translating utility geographic data into power flow models engineers can actually run
- **📐 Data modeling** — clean, typed, interoperable schemas for grid assets
- **🤖 AI integration** — using LLMs to make grid data accessible without needing a PhD to query it
- **🖥️ Full-stack apps** — tools that researchers and operators can actually use
